\doxysubsection{Detailed Description}
Holds all GPU and rendering information needed for rendering a mesh. 

This struct is created by Mesh3\+D-\/derived classes and used by the Render\+System to issue draw calls with the appropriate LOD, shader pipeline, and textures.
