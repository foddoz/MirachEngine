\doxysubsection{Detailed Description}
Interface class defining all enemy state transition logic. 

This interface is designed to separate the behavior of different enemy states (\doxylink{class_attack}{Attack}, \doxylink{class_flee}{Flee}, \doxylink{class_patrol}{Patrol}, \doxylink{class_chase}{Chase}, \doxylink{class_death}{Death}, \doxylink{class_pain}{Pain}) into clearly defined {\ttfamily Enter}, {\ttfamily Execute}, and {\ttfamily Exit} methods. Implementing classes should define these methods for each state.
