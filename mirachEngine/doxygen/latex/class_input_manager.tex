\doxysubsection{Detailed Description}
Manages input actions, key states, and mouse interactions at a high level. 

The \doxylink{class_input_manager}{Input\+Manager} maps abstract Input\+Actions to physical key codes and tracks key/mouse states across frames to support features like "{}just pressed"{} detection.
