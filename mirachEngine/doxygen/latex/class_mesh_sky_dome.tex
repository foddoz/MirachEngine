\doxysubsection{Detailed Description}
A hemispherical skydome mesh used to render sky backgrounds. 

This class generates a dome-\/shaped mesh with configurable resolution and radius, uploads it to GPU, and prepares it for ECS rendering. It supports texture mapping for sky visuals (e.\+g., clouds or gradients).
