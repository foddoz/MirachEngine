Core ECS-\/based rendering system. 

Iterates over all entities that contain a \doxylink{struct_mesh_renderer}{Mesh\+Renderer} and \doxylink{class_transform}{Transform} component, and renders each using its assigned shader pipeline and transform. Applies frustum culling using the provided camera, binds the appropriate vertex arrays and textures, and issues draw calls.
