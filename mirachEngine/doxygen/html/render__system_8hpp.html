Renders a skybox or skydome mesh as a background element.

This should be called before rendering opaque geometry. It disables depth writing to ensure it is always behind other objects.

Parameters

void RenderSystem ( const Program & gApp,
const Entity & camEntity,
EntityManager & entityManager )

Core ECS-based rendering system.

Iterates over all entities that contain a MeshRenderer and Transform component, and renders each using its assigned shader pipeline and transform. Applies frustum culling using the provided camera, binds the appropriate vertex arrays and textures, and issues draw calls.

Parameters
gAppReference to the main Program instance holding global GPU state.
camEntityThe entity containing the active camera component.
entityManagerReference to the ECS EntityManager for accessing components.
