Detailed Description

Handles generation, GPU upload, and ECS/physics registration of terrain chunks.

This class is responsible for dividing a large heightmap into chunks, generating mesh data for each chunk, uploading it to the GPU, and registering each chunk with the ECS and physics systems for rendering and collision.
