Detailed Description

Represents a sub-section of a larger terrain for LOD or streaming.

Inherits from MeshTerrain and extracts its data from a global heightmap. This allows terrain to be split into smaller pieces for performance and culling.

Generates mesh data for this chunk from a global heightmap source.

Generate terrain chunk mesh from a portion of a global heightmap.

This function extracts a rectangular region of height values from a larger terrain and constructs vertex/index buffers only for that portion.

Parameters
globalHeightmapThe full terrain heightmap shared across all chunks.
startXThe X-coordinate (column) in the global heightmap to start this chunk.
startZThe Z-coordinate (row) in the global heightmap to start this chunk.
chunkSizeThe number of vertices in width and depth this chunk should span.
globalWidthThe full width of the global heightmap (used for indexing).
minHeightMinimum expected height value (used for AABB normalization).
maxHeightMaximum expected height value.
textureScaleUV scaling factor for this chunk's texture coordinates.
textureFileNamesA list of texture file paths to use on this chunk.
