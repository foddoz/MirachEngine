\doxysubsection{Detailed Description}
\subsubsection*{template$<$class T$>$\newline
class Singleton$<$ T $>$}
A generic \doxylink{class_singleton}{Singleton} template class. 

This template provides a thread-\/safe and lazy-\/initialized singleton instance. The constructor and assignment operators are deleted to enforce a single instance.
