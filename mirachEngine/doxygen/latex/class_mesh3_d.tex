\doxysubsection{Detailed Description}
Abstract base class for rendering and managing 3D mesh data. 

{\ttfamily \doxylink{class_mesh3_d}{Mesh3D}} defines a common interface for all 3D meshes in the engine. It encapsulates Open\+GL vertex/index buffers, bounding volume, shader pipeline, and associated texture data. Derived classes must implement data upload and rendering conversion.
