\doxysubsection{Detailed Description}
Handles image loading, GPU uploading, binding, and cleanup of texture resources. 

This class provides an abstraction over Open\+GL texture operations. It supports 2D textures and wraps functionality for loading from file, uploading to GPU memory, and binding to specific texture units.
