\doxysubsection{Detailed Description}
Represents a single Level of Detail (LOD) variant of a mesh. 

Each LOD stores its own VAO, VBO, and IBO, along with index count and a local \doxylink{struct_a_a_b_b}{AABB}. Used for LOD switching in rendering systems to improve performance based on distance.
