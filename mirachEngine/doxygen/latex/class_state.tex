\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class State$<$ T $>$}
Abstract base class representing a state in a finite state machine (FSM). 

This class defines the interface for states that can be used within a \doxylink{class_state_machine}{State\+Machine}. Each state should override the Enter, Execute, and Exit methods to define its behavior.
