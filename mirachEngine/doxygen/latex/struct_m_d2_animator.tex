\doxysubsection{Detailed Description}
ECS component for animating an entity with an MD2 mesh. 

Stores the current animation state for an entity and performs per-\/entity frame interpolation. Each entity shares a reference to a read-\/only MD2 mesh and owns its own animated vertex buffer.
