\doxysubsection{Detailed Description}
A physics collision callback that applies damage to entities hit by a thrown object. 

This callback is registered when an object is thrown. It checks for collisions, and if the thrown object hits a robot with sufficient speed, it reduces the robot\textquotesingle{}s health.
