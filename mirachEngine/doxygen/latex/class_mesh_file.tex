\doxysubsection{Detailed Description}
Represents a static 3D mesh loaded from file (e.\+g. OBJ, FBX, etc.) using Assimp. 

Handles loading vertex and material data from external model files, converting them into Open\+GL-\/compatible buffers, and exposing rendering interfaces.
