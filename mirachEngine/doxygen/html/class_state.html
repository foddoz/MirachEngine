Detailed Description

template<typename T>
class State< T >

Abstract base class representing a state in a finite state machine (FSM).

This class defines the interface for states that can be used within a StateMachine. Each state should override the Enter, Execute, and Exit methods to define its behavior.

Template Parameters
TThe type of the owner entity using this state.
