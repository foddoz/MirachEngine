\doxysubsection{Detailed Description}
System class responsible for updating all \doxylink{class_enemy}{Enemy} entities. 

This system queries all entities that have an \doxylink{class_enemy}{Enemy} component and updates them each frame by calling their individual {\ttfamily \doxylink{class_enemy_system_af199ce30b80387110b68299836f760cf}{Update()}} methods. It is designed to work with an ECS (Entity-\/\+Component-\/\+System) framework.
