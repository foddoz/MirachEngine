\doxysubsection{Detailed Description}
A 3D animated mesh class for rendering MD2 models with frame interpolation. 

Inherits from \doxylink{class_mesh3_d}{Mesh3D}. Supports loading MD2 models, playing animations by name, and interpolating between frames for smooth animation playback.
